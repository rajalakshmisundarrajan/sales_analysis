📊 Sales Data Analysis Project

🚀 Project Overview

This project analyzes sales data using Python and Pandas to extract meaningful business insights such as revenue, top-performing products, regional sales distribution, and customer trends.

The goal is to demonstrate data cleaning, analysis, and reporting skills in a structured and scalable way.


📁 Project Structure

│── sales_data.csv         # Dataset
│── analysis_report.md     # Detailed report

📊 Dataset Description

The dataset contains the following columns:

- Date → Transaction date
- Product → Product name
- Quantity → Units sold
- Price → Price per unit
- Customer_ID → Unique customer ID
- Region → Sales region
- Total_Sales → Revenue per transaction

---

🔍 Analysis Performed

🔹 Total Revenue

Sum of all sales transactions.

🔹 Best Selling Product

Product generating the highest revenue.

🔹 Average Sales

Average revenue per transaction.

🔹 Sales by Region

Total revenue grouped by region.

🔹 Monthly Sales Trend

Sales grouped by month to identify trends.

🔹 Top Customers

Top 5 customers based on purchase value.
